# Examen Interactivo de Marketing Estratégico y Operativo

Examen de práctica autocorregible sobre los **Temas III, IV, V y VI**: segmentación, targeting, posicionamiento, producto, precio, distribución y comunicación.

## 👉 Hacer el examen

**➡️ https://nissanguy365.github.io/examen-marketing/**

Abre el enlace en cualquier navegador (móvil u ordenador) y empieza el examen.

## Contenido

- **Sección 1 — Verdadero / Falso** (10 preguntas)
- **Sección 2 — Selección múltiple** (10 preguntas, 4 opciones)
- **Sección 3 — Desarrollo** (3 preguntas con respuesta modelo)

## Características

- ⏱️ Temporizador de 45 minutos
- 📊 Barra de progreso
- ✅ Corrección automática de las secciones 1 y 2 con respuestas resaltadas
- 📝 Respuestas modelo para la sección de desarrollo

Todo el examen está contenido en un único archivo `index.html` (sin dependencias externas).
